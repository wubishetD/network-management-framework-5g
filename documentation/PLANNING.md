## Project Planning

The project is structured into six milestones covering the full lifecycle from
planning and design to implementation, testing, and final demonstration.

Each milestone consists of multiple work packages that are tracked as GitHub
Issues and organized using the GitHub Project board.

### Project Roadmap

The overall timeline, milestones, and progress can be viewed in the GitHub
**Roadmap view**:

➡️ **[View Project Roadmap](https://github.com/orgs/MobileComputingWiSe25-26/projects/15)**

The roadmap provides a time-based overview of all milestones and work packages
and reflects the current project status throughout the semester.
