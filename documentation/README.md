## Project Summary

- Mobile Broadband (eMBB)
- Connected Cars / Emergency Communication (URLLC)
- Industrial IoT / Smart City (mMTC)

## Deployment Focus
- Network Slicing using Open5GS
- RAN & UE simulation using UERANSIM / PacketRusher
