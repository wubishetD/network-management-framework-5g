### Collaboration & Communication

- **Meetings:** Weekly team meetings with documented minutes  
- **Communication tools:** GitHub Issues, GitHub Project Board, messaging platform (e.g., WhatsApp/Teams)  
- **Task tracking:** All tasks are tracked as GitHub Issues and linked to milestones  
- **Decision tracking:** Major technical and design decisions are documented in the repository

---

### Contribution Policy

- Every team member contributes to **multiple milestones**
- Work packages may span multiple milestones where technically meaningful
- All contributions are tracked through GitHub commits, issues, and milestone activity
